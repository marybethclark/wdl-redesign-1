WDL SYSTEMS PROUDLY DISTRIBUTES ICOP'S VMXP-6453 SBC

Pittsboro, NC – 13 September 2012 – WDL Systems proudly distributes ICOP's VMXP-6453 Single Board Computer in the PC/104 form factor from ICOP Technology. The VMXP-6453 is powered by the 800MHz Vortex86MX+ SoC which provides over twice the performance of the previous Vortex86DX SoC.

ICOP's VMXP-6453 features a heat sink which also doubles as an EMC shield, 4x USB 2.0, 2x RS-232 ports, 16-bit GPIO, HD audio, on-board PWM, up to 1GB DDR2 RAM and up to a 4GB EmbedDisk (SST Flash Disk) are available onboard options.

VMXP-6453 Features Include:
 • DM&P SoC CPU Vortex86MX+ @ 800MHz
• 1GB DDR2 onboard
• Watchdog Timer, software programmable, x 2 sets
• 2D VGA chip, Share system memory 16MB/32MB
• ALC 262 HD Audio, 10/100Mbps Ethernet
• Operating Temp: 0°C ~ +60°C
• Measures 90 x 96mm (3.54 x 3.77 inches)
