WHITE DWARF LINUX Continues To Make Life Easier for Developers
- Version 1.1 provides support for a wide range of SBCs

Apex, NC, 19 March 2001 - WDL Systems, a leading distributor of embedded solutions, is pleased to announce the release of white dwarf linux, version 1.1. Visit www.whitedwarflinux.org

wd linux version 1.1 upgrades the tiny distributon of Linux that developers are finding is ideal for Internet appliances, process control, medical equipment, industrial automation, wireless applications and testing equipment. It offers support for any motherboard or single board computer with IDE support and at least 8MB of RAM. It also supports both CD and network installs.

white dwarf linux contains more than 40 software packages, including a web browser, an Apache 1.3.x web server and email capability. In version 1.1, the installer has been moved to a modular kernel on the bootdisk, which makes for easier support for future platforms. It has also been rewritten to handle both CD and network installs. Network installs require 16MB of RAM; CD installs require as little as 8MB of RAM.

In addition, beta support for glibc has been added. glibc 2.1.x binaries can be used with white dwarf without having to recompile the older libc5 libraries. This allows easy copying of binaries from Slackware 7.x and RedHat 6.x with no changes to white dwarf. Full support will be included in version 1.2.

A web site has been developed for the installation and support of version 1.1. Visit www.whitedwarflinux.org

white dwarf linux was initially developed for Adastra/JUMPtec (now known as Kontron) single board computers. It works with the DIMM-PC 486, MOPSlcd4 and MOPS586, MOPS686+ and MOPSlcd6 and the CoolMONSTER boards (see below). It has also been successfully tested with the Tri-M MZ104 board which supports the ZF Micro Devices MachZ chip.

KONTRON Single Board Computers
The DIMM-PC 486 is a tiny, fully functioning PC that measures only 40x68mm (1.57 x 2.68 inches). It features an AMD 486 66MHz processor with 16MB of RAM and 16MB of Flash, and onboard support for an IDE drive, floppy drive, 2 serial ports, 1 parallel port, keyboard and programmable watchdog timer. Support for VGA/LCD, Ethernet and additional serial ports are provided through separate I/O modules.

The MOPS boards from Kontron are true PC/104 boards that support onboard VGA/LCD, Ethernet, 2 serial ports, 1 parallel port, IDE, floppy and keyboard. The MOPSlcd4 and MOPS586 feature AMD 586 133MHz processors, 4MB of RAM, 1.9MB of Flash. The MOPS686+ and MOPSlcd6 feature Intel® Pentium® 166MHz or 266MHz processors and 10/100 Base T Ethernet, USB and PC/104-plus bus.
