WDL SYSTEMS NOW SHIPPING M-5360 ARM CORTEX A8 SOM BY ARTILA

Pittsboro, NC - 1 November 2013 - WDL Systems is now shipping the M-5360A ARM Cortex A8 System On Module (SOM) from Artila, a leading designer and manufacturer of ARM based embedded computers.

M-5360A is a credit card size SOM powered by Freescale 800MHz i.MX537 ARM Cortex A8 processor with 1GB DDR3 RAM and 4GB eMMC Flash. M-5360A features two independent LVDS channels for dual LCD displays and one VGA port for external monitor connection.

The multimedia and graphic engine of i.MX537 supports OpenGLE 2.0, OpenVG 1.1 graphics acceleration and 1080P video decoding. In addition to the multimedia features, M-5360A also provides powerful communication functionality such as Ethernet, RS-232, RS-485, CAN 2.0, 1-Wire and USB making the M-5360A suitable for multimedia applications as well as embedded networking.

M-5360A uses reliable 128-pin 2.0 mm pin headers which enable easy design of an application board. The operating system supported and pre-installed can be Linux, Android and Windows CE. In addition to the hardware building blocks, plenty of software utility and device drivers are available for users’ applications.

M-5360A Features Include:
• Freescale i.MX537 Cortex A8 800MHz
• 1GB DDR3 RAM
• 4GB eMMC Flash
• 24-bit LCD LVDS x2
• Analog VGA x1
• Support full HD 1080P video decode
• Support OpenGLE 2.0 & Open-VG 1.1
• 10/100 Mbps Ethernet x1 with Phy
• UART ports x4
• USB 2.0 host and USB OTG
• SATA,SD,SPI,I2C,I2S,1-Wire,GPIO and CAN 2.0
• OS: Linux, Android and Window Compact 7.0
