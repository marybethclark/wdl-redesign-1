EOS-4000
Compact Vision System

EOS-4000 Series by ADLINK is a rugged, highly compact embedded vision system with 3rd Generation Intel® Core™ i5/i7 processor, dual independent PoCL (power over Camera Link®) ports with data transfer up to 2.56 Gb/s, and pixel clock rates up to 85MHz for high speed capture of large images.

FEATURES :
• Embedded 3rd Gen Intel® Core™ i7 Quad Core CPU
• 2-CH PoCL, base Camera Link® configuration
• One internal USB port and 1kbit EEPROM
• RAID system (Dual SATA interface)
• 64-CH isolation DI/O with digital filter
• Hardware monitoring and watchdog timer

Digital I/O
USB DAQ

USB-7230/7250 Isolated USB Digital I/O Modules by ADLINK are Standalone USB DAQ modules featuring high voltage on/off control and monitoring, and isolation voltage supported up to 2500VRMS.

FEATURES :
• USB 2.0, USB bus powered
• Programmable digital filter for input accuracy
• Programmable DO/Relay initial status
• 2-CH frequency/event counters
• Up to 2500 VRMS isolation voltage
• Ready-to-use testing application (U-Test) provided
• 32-CH isolated digital I/O (USB-7230)
• 8-CH solid-state relay output, 8-CH isolated DI (USB-7250)
