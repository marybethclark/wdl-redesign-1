WDL Systems Distributes Artila's Web Based Remote I/O Solutions

Pittsboro, NC - 8 December 2015 - WDL Systems proudly distributes Artila’s RIO-2010 Modbus / TCP Remote Digital IO Module Solution for Machine-to-Machine (M2M) Sensors and Actuators.

Artila's enhanced RIO-2010, is the new generation of web enabled data acquisition and control module that is ideal for remote monitoring applications. RIO-2010 supports MQTT, a lightweight publish / subscribe messaging protocol for the OASIS standard that is designed for sensor and device communication with low network bandwidth.

RIO-2010 is equipped with MAXIM 1-wire bus for acquiring temperature data from a DS18B20 digital thermometer, 16 isolated digital input channels for monitoring high / low status and 8 relays to perform On/Off control.  RIO-2010's built-in web server is used to configure alarm settings, On/Off control strategy and email alert.  Once alarmed, RIO-2010 sends a message to the MQTT broker, logs the event, triggers relay on or off and then sends an email to the administrators.

In addition to MQTT, RIO-2010 also supports Modbus TCP / RTU control and Web AJAX interface for flexible and easy system integration.  Management utility software is available for users to configure network and MQTT, authorized IP and Modbus settings.

RIO-2010 Features:

  • Web based remote I/O monitoring and control
  • Web server and web interface ready
  • Modbus, AJAX, MQTT
  • Stand-alone alarm relay control and email alert
  • Periodically data and event publishing to server (broker) via MQTT
  • Form A or form B relay with contact rating 30VDC@1A or 125VAC@0.5A
  • 16 isolated digital input and 1-wire temperature measurement
