mSATA
Industrial SSD

Swissbit X-200m Industrial mSATA Solid State Drives (SSD) are highly reliable data storage drives with an internal intelligent controller to manage interface protocols, data storage and retrieval as well as hardware BCH-code Error Correction Code (ECC), defect handling, diagnostics and clock control. The drive has a voltage detector and a powerful power-loss management feature to prevent data corruption after power-down.

FEATURES:
• mSATA Form Factor: 50.8mm x 29.85mm x 3.3mm
• High reliability : MTBF > 2,500,000 hrs
• 2GByte up to 64GByte (SLC NAND Flash)
• S.M.A.R.T. support
• Extended Temperature: -40°C ~ +85°C
• Life Cycle Management
• RoHS compatible
