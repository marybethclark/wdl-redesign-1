Condition Monitoring
At-a-Glance

The IMON-U310 Industrial 3G Router for Condition Monitoring from INSYS icom is an intelligent M2M solution for the latest Internet of Things (smart IoT) applications. The IMON-U310 is equipped with a monitoring app that enables automatic status notifications (text message, e-mail or SNMP) that can be configured via logical links; and can be used to trigger pre-defined actions, such as starting or stopping a controller or switching an output.

FEATURES:
• Professional, industry-grade VPN 3G router
• Various interfaces: RS-232, RS-485, RJ-45, digital IOs
• Integrated processing power (Linux sandbox)
• Monitoring app for a variety of PLCs and other devices
• Messages on defined actions via SMS, e-mail and http call
• Direct connection to cloud services like Cumulocity

DragonFly™
for IoT Applications

MultiConnect® Dragonfly™ embedded cellular SoMs from MultiTech are fully certified and MNO approved, ready-to-integrate modules that offer developers the functionality of a programmable computing platform with the convenience of an onboard cellular radio all in one compact design.
