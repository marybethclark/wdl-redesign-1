WDL SYSTEMS NOW SHIPPING MATRIX-513 ARM9 BOX PC BY ARTILA

Pittsboro, NC - 10 March 2014 - WDL Systems is now shipping the Matrix-513 ARM9 400MHz CPU ATMEL9G45 + Linux 2.6.38 Wireless Box Computer by Artila Electronics. The Matrix-513 is a powerful computing platform for applications requiring remote device management and reliable wired and wireless communications plus low power.

Matrix-513 comes with two MiniPCIe slots which can accept COTS Wireless LAN, Bluetooth, GPS and cellular 3G MiniPCIe cards. The dual independent 10/100 Mbps Ethernet ports of Matrix-513 provide a flexible and reliable wired communication and the four RS-232/422/485 serial ports are ready for connecting to the serial devices. In addition, Matrix-513 is equipped with two isolated digital inputs and one relay output which can be used for device status monitoring and alarm.

Matrix-513 is shipped with Linux 2.6.38 kernel and file system plus many software utilities such as web server, PHP, python, java, MySQL and SQLite. The software development tool chain GCC C/C++ is also ready in the Artila CD and website for free download.

Matrix-513 is powered by 400 MHz Atmel AT91SAM9G45 ARM9 SoC and 128MB DDR2 RAM and 256MB NAND Flash. A 2MB DataFlash is used as backup file system and It provides users an easy way to perform system update or recovery at the service site.

Matrix-513 Features Include:

  • Linux 2.6.38 computing platform with file system
  • CPU: ATMEL AT91RM9G450, 400MHz
  • Flash: 256MB, NAND type
  • RAM: 128MB DDR2
  • micro-SD socket: 1x inside, up to 32GB max.
  • Ethernet: 2x, 10/100Mbps
  • Serial Ports: 4x RS-232/422/485
  • USB 2.0 Hosts: 2x, up to 480Mhz
  • One full-size mini-PCIesocket inside
  • One half-size mini-PCIesocket inside
  • Power input: 9~48VDC, 500mA@12V
  • GNU C/C++ tool chain for Linux and Windows
