WDL SYSTEMS NOW SHIPPING ADLINK MXE-5300

Pittsboro, NC - 31 July 2013 - WDL Systems offers the M-9G45A Linux-Ready ARM System On Module (SOM) from Artila Electronics featuring remote desktop control support for Linux and WinCE.

The credit card sized M-9G45A is powered by an Atmel 400MHz AT91SAM9G45 ARM9 Processor with an onboard memory management unit, and is equipped with 128MB DDR2 RAM, 128MB NAND Flash, and 2MB DATA Flash. Perfect for Human Machine Interface (HMI) devices, the M-9G45A is also an ideal solution for M2M device management.

The M-9G45A supports TightVNC on the Linux model and Remote Display Control on WinCE and is equipped with 4x UART ports, 10/100Mbps Ethernet, GPIO, SPI, I2C and I2S bus and high speed USB 2.0 port. Wireless communication is ready to go by using commercial off the shelf USB WiFi and 3G dongle.

M-9G45A Features Include:

• Linux kernel 2.6.38 with file system
• CPU: ATMEL AT91SAM9G45, 400MHz
• Flash: 128MB, NAND type
• SDRAM: 128MB
• LCD interface: TTL(24-bit), upto 1280 x 860 pixels
• 10/100Mbps Ethernet
• USB 2.0
• SD card (32GB max.)
• UART: 4x 921.6kbps w/ hardware flow control
• SPI interface: 2x chip select signals
• GPIO interface: 15x pins, CMOS/3.3V compatible
• Dimensions: 80mm x 50mm
• Power input: 80mA@3.3V
• GNU C/C++ tool chain for Linux and Windows
