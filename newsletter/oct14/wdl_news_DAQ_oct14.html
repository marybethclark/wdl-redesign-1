PCI Express
Digital I/O

PCIe-IDIO-24 Boards from ACCES I/O offer x1 lane PCIe isolated digital input and FET output with Change of State (COS) detection capabilities. In addition to the isolated signals, 8 (or 4) TTL/CMOS input/output pins are available.

FEATURES :
• Optically isolated, non-polarized digital inputs
• Software configurable filters on inputs
• Can detect input state change and assert interrupt
• 24 optically isolated high-side FET switches
• Four optically isolated output groups and two
• Optically isolated input groups
• Opto-couplers rated for 2.5kV isolation
• 8 non-isolated TTL/CMOS I/O lines
