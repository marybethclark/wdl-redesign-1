'SPARK YOUR IMAGINATION' WITH ICOP TECHNOLOGY AND WINDOWS EMBEDDED CE

Pittsboro, NC – 12 May 2008 – WDL Systems is now shipping Jumpstart SPARK Kits featuring the eBox-2300SX and eBox-4300 from ICOP Technology and Windows Embedded CE 6.0 R2. Each Jumpstart SPARK Kit contains either ICOP's eBox-2300SX or eBox-4300 low-power small-footprint compact computing device with a full version of Windows Embedded CE 6.0 R2 and Visual Studio 2005 Professional, a software offering previously available only to professionals.

“The SPARK Your Imagination promotion is a joint offering between ICOP and Microsoft Windows Embedded, and WDL Systems is proud to be a distributor of this technology,” says WDL Systems president, Mark Estill. “Jumpstart SPARK Kits combine embedded hardware, development tools, technologies and resources, allowing individuals and small groups to harness the creativity of Windows Embedded CE.”

Each Jumpstart SPARK Kit comes with either the eBox-2300SX-MSJK or eBox-4300-MSJK and the following:
• 256MB IDE bootable flash storage preconfigured with DOS boot-loader and working CE 6.0 image.
• Full version of Microsoft Windows Embedded CE 6.0 R2 Platform Builder.
• Full version of Microsoft Visual Studio 2005 Professional.
• Board-Support-Package, SDK and pre-built Windows Embedded CE 6.0 image.
• RJ-45 Crossover Ethernet & Null Serial modem cable, use to link eBox-2300SX to Windows Embedded CE 6.0 Platform Builder development station for image transfer and debug.

For more information and to order Jumpstart SPARK Kits, please visit our Spark Your Imagination web page.
