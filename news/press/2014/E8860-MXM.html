WDL SYSTEMS NOW SHIPPING TUL's E8860 MXM GRAPHICS CARDS

Pittsboro, NC - 20 November 2014 - WDL Systems is now shipping Tul’s E8860 MXM High Performance Graphics Card. The E8860MF Graphic Card from Tul Embedded is a MxM 3.0 Module graphics card using the standard MXM A type form-factor. The E8860MF features a customizable V-Bios for display output requirements of mobile devices, panel PCs and server product applications with a flexible and convenient upgrade solution.

Tul's E8860MF uses the embedded E8860 Graphics chip from AMD and incorporates 128-bit 2GB GDDR5 memory in MCM BGA packaging. Industrial grade passive electronic components are used throughout the entire card and it has been tested between 0 ~ 50Â°C to ensure ideal system compatibility and reliability for industrial and automation industry applications. The core AMD E8860 chip has a product supply cycle of up to 5 years to ensure a stable lifecycle for industrial system and embedded products while reducing development costs.

E8860 MXM Features Include:

  • AMD E8860 128-bit, 2GB GDDR5
  • MXM 3.0 Type A
  • PCI-Express 3.0
  • Supports DX11.1, OpenGL 4.1+, OpenCL 1.2+
  • Supports Shader Model 5.0
  • Optional Fan solution
