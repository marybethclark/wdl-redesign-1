USB-3488A HIGH PERFORMANCE GPIB INTERFACE CONTROLLER from ADLINK

Pittsboro, NC – 15 August 2009 – WDL Systems is proud to distribute USB-3488A, the High-Performance IEEE-488 USB GPIB Interface Controller from ADLINK. The USB-3488A GPIB interface provides a direct connection between the USB port on a desktop or laptop computer to GPIB instrumention. With the USB-3488A GPIB interface and its USB Plug and Play feature, GPIB instruments can be connected and disconnected without having to shut down the computer. No external power supplies are necessary. The USB-3488A GPIB interface is equiped with a 2 meter USB cable that is USB 2.0 compliant.

“The USB-3488A GPIB from ADLINK is compatible with existing applications and requires no re-writing of code,” says Donnie Springfield, technologist for WDL Systems. “The Plug and Play USB-3488A GPIB is low power, lightweight and highly cost efficient.”

ADLINK's GPIB interface solutions are delivered with complete software support, including a driver API set that is fully binary compatible with NI-488.2• driver. All programs written based on the GPIB-32. DLL library can be executed with the USB-3488A without any modification. The VISA library is also supported to ensure compatibility with applications utilizing VISA. Whether using VC++, VB, Delphi, LabVIEW•, or any other T&M ADEs, the ADLINK USB-3488A provides Plug and Play compatibility with all existing applications. •(NI, LabVIEW, CVI and other names are trademarks or propriateries of National Instruments Corporation or other companies.)

Integrated features of the USB-3488A GPIB include:

• Easy GPIB connectivity for laptop computers
• Plug and Play interface; No GPIB cable required
• APIs compatible with the NI-488.2• driver software
• IEEE 488.1 and 488.2 compatible
• Full compatibility with industry-standard VISA library
• USB 2.0 compatible, 2m USB cable attached
• Maximum GPIB transfer rates of more than 1.2 MB/s
• No external power required
