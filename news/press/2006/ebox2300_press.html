WDL SYSTEMS OFFERS A SUB $100 EMBEDDED PC SOLUTION WITH EBOX-2300

Pittsboro, NC – 5 December 2006 – WDL Systems is pleased to announce the release of a sub $100 embedded computer that ships with a case and power supply. The eBox-2300 from ICOP Technologies is a small yet full-featured 200MHz computer for the embedded designer that sells for a remarkable price.

The eBox-2300 is an embedded PC built on a 200Mhz Vortex86 System-On-Chip technology. It comes with 128MB system memory, integrated Audio, Ethernet, VGA and USB. The eBox-2300 is ideal for limited physical space. The dimensions of the eBox-2300 are 115mm x 115mm x 35mm (less than 5” square by 2” deep). The VESA mounting holes allow for easy integration with a VESA LCD allowing computer access virtually anywhere. The x86-based hardware is compatible with Windows CE and Windows XP Embedded, Windows CE Platform Builder, and Windows XP Embedded.

“The eBox-2300 is a great tool for embedded developers of every experience level. It fits into any budget, and can be adapted for many uses,” Mark Estill, president of WDL said. “The eBox-2300 is revolutionizing the Embedded PC market.”

For more information and to order the eBox-2300 from WDL Systems online, please visit our website's ICOP eBOX EmbedBox Computer section.

eBox-2300 Features Include:
 - 200MHz Vortex86 System-On-Chip
- 128MB SDRAM
- 44-Pin IDE interface
- Supports EmbedDisk bootable IDE flash storage
- PXE diskless boot
- Video with 8MB shared system memory
- 15-pin D-type VGA output
- Support 640x480, 800x600, 1024x768 and 1280x1024 display
- Type I/II Compact Flash slot
- Mini-PCI Expansion Slot (Optional)
- 2 RS-232 Serial Ports (Optional)
- 3 USB 1.1 Ports
- Realtek 8100B 10/100 Mbps Ethernet with RJ-45 connector
- PS2 Keyboard and Mouse
- AC97 Audio with Mic-in and Line-out
- Single +5V voltage with ACPI function support
- 0 ~ +60 degree C operating temperature
- Dimension: 115mm x115mm x 35mm
- VESA mounting holes
- Ultra low power (15 Watts)
